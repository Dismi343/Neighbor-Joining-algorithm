# Neighbor-Joining Algorithm Implementation

A Java implementation of the Neighbor-Joining algorithm for constructing phylogenetic trees from distance matrices.

## Files
Consists of 2 file named Njalgo.java and NewickTree.java
From them the Njalgo.java file consists the main algorithm I used for the Neighbor-Joining algorithm
Using the file NewickTree.java (which was taken my stackoverflow) the final Tree visalisation is created.

From the original source code the file NewicjTree.java was changed, the input variable type to double from integer. And I have created a seperate method to visualize the tree, as author hasn't created a method to create a visualization. 


## How to run the code
compile = javac Njalgo.java
run = java Njalgo

(No need of compiling and running the NewickTree.java file.)
